OnlineFoodDeliverySystem - Console Project (v2)
==============================================
How to run in VS Code:
1. Unzip the project.
2. Open folder in VS Code.
3. Install 'Java Extension Pack' if not installed.
4. Run src/com/onlinefood/app/App.java using Run > Run Java or using terminal:
   cd src
   javac com/onlinefood/app/App.java
   java com.onlinefood.app.App
Features:
 - Admin Menu: add restaurant, add food item, remove food item, view restaurants, view orders, add delivery person, assign delivery person.
 - Customer Menu: register, view restaurants, add items to cart, view cart, place order.
Notes:
 - Data is in-memory. Restart clears data (except seed data).
 - Menu prompts match the sample output style from your case study PDF.
